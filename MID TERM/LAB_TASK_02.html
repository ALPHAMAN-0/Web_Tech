A.3 - Description List: Web Technologies

HTML
HyperText Markup Language is the standard markup language used to create web pages. It provides the structure and content of web documents.
CSS
Cascading Style Sheets is used for styling and layout of web pages. It controls colors, fonts, spacing, and responsive design.
JavaScript
JavaScript is a programming language that adds interactivity and dynamic behavior to web pages. It runs in the browser and enables user interactions.

D.1 - Demonstrating Global HTML Attributes

Global Attributes Used in This Page:

1. ID Attribute

The id attribute uniquely identifies HTML elements. Examples in this page:

  • id="header-section" - Identifies the header
  • id="part-a" - Used for anchor links to jump to sections
  • id="featured-image" - Identifies the featured image container
  • id="global-attr-demo" - Identifies this section
2. CLASS Attribute

The class attribute allows multiple elements to share the same style. Examples:

  • class="main-header" - Styles the main header
  • class="content-section" - Styles all major sections
  • class="programming-list" - Styles the programming languages list
  • class="internal-link" - Styles internal navigation links
3. TITLE Attribute

The title attribute provides a tooltip when hovering over elements. Examples:

  • title="Visit American International University of Bangladesh" - On the AIUB link
  • title="Web Development - HTML, CSS, JavaScript" - On the image
  • title="This section demonstrates HTML global attributes" - On this section
